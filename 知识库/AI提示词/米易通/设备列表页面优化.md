productGroup 值列表：
E50Pro 、 LB20S 、 GT600 、 M50 、 MR702 、 LR200 、 M20 、 E40 、 DR030 、 BHY-3S 、 DAS 、 MR701 、 Other

请你处理 products.json 中的数据，提取 productToken，遵照示例中 json 的格式，将输出的 json 数据保存到 app/src/main/assets/product_config.json 文件中，productGroup 限定为 productGroup 值列表中的值，productIdList 为 id 列表。要求：
1. 即使 productToken 能匹配上 productGroup 值列表中的值，如果 deviceNum 为 null 或 0，表示该产品下没有设备，则此 id 不添加到 productIdList 中
  - 例如 "id": 131, "productToken": "E40S", "deviceNum": null，虽然 productToken 能匹配上 productGroup 值列表中的 E40，但是 deviceNum 为 null，则此 id 不添加到 productIdList 中

2. 注意可能有多种变体或特殊匹配规则
    - E40 和 E40S 归为 E40 产品组
    - MD-LL030、MD-DR030、MD-NW030、MD-LD30 归为 DR030 产品组
    - M20_GNSS、M20_1、M20 归为 M20 产品组
    - BHY-3S 归为 BHY-3S 产品组
    - productToken 以 DAS 为前缀归为 DAS 产品组  
    - productToken 以 MR701 为前缀或 productName 以 MR701 为前缀归为 MR701 产品组
    - ADME、ADME_HAC10、MR501 、MD-RN0X 归为 Other 产品组
    - 其余剩下的产品不处理
